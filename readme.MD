# VPC Module 
This module is devloped for joindevops company. projects inside joindevops follows this module to create there vpc 

# Inputs
Project --  Mandatory
users should specify a project name 
environment- mandatory
users should specify a environment 
CIDR - optional 
users should provide cidr block, default is 10.0.0.0/16


# output.tf
vpc_id export vpc id created
